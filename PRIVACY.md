# Privacy

Claude in Codex does not add analytics, remote telemetry, or background data collection of its own.

## What the plugin does

- reads a local JSON spec file
- invokes the locally installed `claude` CLI
- streams delegated output back into the current local workflow

## What the plugin does not do

- it does not operate a hosted backend
- it does not upload project data to a service controlled by this repository
- it does not persist conversation transcripts outside your existing Codex and Claude environments

Any data handling performed by Codex, Claude Code, GitHub, or other tools remains subject to those products and services.
